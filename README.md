ATWSS
=====

Atlas Wireless Safety System (website)

Rebuild the website:

- Make sure you have node.js and npm from nodejs.org
- in build run 
npm install

- to rebuild from root dir run
node build

== Using Jekyl (not in production yet)

yum install ruby
yum install ruby-devel
yum install rubygems
yum install gcc
gem install jekyll
